Module Cps.Rejection_sampling

type prob =
  1. | Prob

Type that will be passed as an argument to every function. This allow us to write models using the same syntax as in the Enumeration and Importance_sampling modules

type ('a, 'b) model = (prob -> 'a option) -> prob -> 'b -> 'a option

Type of a model. Such a model takes a continuation function, a prob object and some data, and returns a value, or None if the execution does not fit the conditions required by the model

val sample : (prob -> 'a -> prob) -> prob -> 'a Distributions.distrib -> prob

draws a sample from a distribution, CPS-style

val assume : (prob -> 'a option) -> prob -> bool -> 'a option

continues the execution if the condition is met, returns None otherwise

val observe : (prob -> 'a option) -> prob -> 'b Distributions.distrib -> 'b -> 'a option

takes a sample from a distribution and continues if the output correspond to the given value, returns None otherwise

val observe_array : (prob -> 'a option) -> prob -> 'b Distributions.distrib -> 'b array -> 'a option

iterates the previous function with an array of observations

val exit : prob -> 'a -> 'a option

returns a value

val exec : ('a, 'b) model -> 'b -> 'a

executes a model until a value is returned

val infer : ?n:int -> ('a -> float) -> ('a, 'b) model -> 'b -> 'a Distributions.distrib

takes a model and infers the law of its output by rejection sampling
