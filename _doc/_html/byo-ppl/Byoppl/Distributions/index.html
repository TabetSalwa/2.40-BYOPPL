Module for distribution data structures

This library contains data structures for basic discrete and continuous distributions.

Types

type 'a prob_law = {
  1. support : 'a array;
  2. probs : float array;
}

Type of a discrete law, with a support being an array of type 'a and their linked probabilities in another array of floats

type 'a distrib = {
  1. sample : unit -> 'a;
  2. pdf : 'a -> float;
  3. logpdf : 'a -> float;
  4. mean : float;
  5. var : float;
  6. law : 'a prob_law option;
}

Type of a distribution data structure. Each distribution has :

  • a sampling function
  • a probability mass or density function
  • its corresponding logarithmic probability mass or density function
  • an analytical mean value
  • an analytical variance value
  • a law if the distribution is discrete

Distributions

Discrete distributions

val bernoulli : float -> int distrib

Bernoulli distribution, taking as argument a certain probability p and yielding

  • a sampling function
  • its probability mass function with \mathbb{P}(0) = 1 - p and \mathbb{P}(1) = p
  • its logarithmic probability mass function
  • the mean \mathbb{E}(X) = p
  • the variance Var(X) = p(1 - p)
  • the law of support \{0;1\} and of probabilities \{1-p;p\} *
val binomial : float -> int -> int distrib

Binomial distribution, taking as arguments a certain probability p, a number of experiments n and yielding

  • a sampling function
  • its probability mass function with \mathbb{P}(X=k) = \binom{n}{k} p^{k} (1-p)^{n-k}
  • its logarithmic probability mass function using the Owl_stats dedicated function
  • the mean \mathbb{E}(X=k) = np
  • the variance Var(X=k) = np \times (1-p)
  • and no law even if discrete because of the possibility of having a very large array

val uniform_discr : int -> int -> int distrib

Uniform distribution on a discrete support, taking as arguments the bounds a and b and yielding

  • a sampling function
  • its probability mass function with \mathbb{P}(X=x) = \frac{1}{b-a+1}
  • its logaritmic probability mass function
  • the mean \mathbb{E}(X) = \frac{a+b}{2}
  • the variance Var(X)=\frac{(b-a)(b-a+2)}{12}
  • the law of support \{a;a+1;...;b-1;b\}

Continuous distributions

val uniform : float -> float -> float distrib

Uniform distribution, taking as arguments the bounds a and b and yielding

  • a sampling function
  • its probability density function with \mathbb{P}(X=x) = \frac{1}{b-a}
  • its logaritmic probability density function
  • the mean \mathbb{E}(X) = \frac{a+b}{2}
  • the variance Var(X)=\frac{(b-a)^{2}}{12}
val exponential : float -> float distrib

The Exponential distribution is the probability distribution of the time between events in a process in which events occur continuously and independently at a constant average rate. This function takes as argument the parameter \lambda and yields

  • a sampling function
  • its probability density function with \mathbb{P}(X=x) = \lambda e^{-\lambda x}
  • its logarithmic probability density function
  • the mean \mathbb{E}(X) = \frac{1}{\lambda}
  • the variance Var(X) = \frac{1}{\lambda^{2}}
val normal : float -> float -> float distrib

Normal distribution, taking as arguments the mean \mu and the standard deviation \sigma, and yielding

  • a sampling function using the Owl_stats dedicated function
  • its probability density function with \mathbb{P}(X=x) = \frac{1}{\sigma \sqrt{2 \pi}} e^{- \frac{1}{2} \left(\frac{x-\mu}{\sigma} \right)^{2} }
  • the mean \mathbb{E}(X) = \mu
  • the variance Var(X) = \sigma^{2}
